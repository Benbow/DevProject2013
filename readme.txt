This repo is for the supinfo dev project 2013 - B3

Notre projet, Farm Warfare, est un jeu de gestion de ferme, type "farmville" qui associe aux aspects 'gestion' bien connut
une dimension MMORPG, tel que les alliances, ou encore un systeme de combat..

Ce projet nous a ete demande en tant que projet de fin d'annee 2013.

C'est donc un MMORPG gestion/combat, entierement developpe en HTML/Javascript. Il utilise le moteur isometrique 3D
pp3diso disponible ici http://www.prelude-prod.fr/demo/pp3diso/ , ainsi que la technologie serveur nodejs !
